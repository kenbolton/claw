# claw | The "Terminal Grip"
A claw gripping a terminal cursor — the tool that holds the agent and drives it.

## Design logic:
- **The Visual**: A lobster claw rendered as two curved strokes, pinching a blue terminal cursor block.
- **The Palette**: Lobster Red (#E2725B) for the claw; Go Blue (#00ADD8) for the cursor/action.
- **The Concept**: The CLI is the grip — it holds, directs, and releases agents across any architecture.

## CLI Brand Palette
If you are building a website or colorful terminal output, use these Hex codes to keep the "Claw Family" consistent:
- **Lobster Red**: #E2725B (Primary branding)
- **Go Blue**: #00ADD8 (Logic & Action)
- **Shell Grey**: #D1D1D1 (Inactive/Legacy states)
- **Deep Sea**: #1E1E1E (Backgrounds/Terminal)

## Recommendation for Implementation
- **Iconography**: Use the "Terminal Grip" for the claw binary icon and GitHub avatar.[1]
- **Visual Language**: On your website or READMEs, use a "layered" illustration style where the claw icon is shown "holding" various shell icons (Nano, Zepto, etc.) to show versatility.
- **Molt Animation**: If you ever do a landing page, a simple SVG animation of the "Double Shell" where the agent moves from the wireframe shell into the solid one would be a powerful visual hook.
